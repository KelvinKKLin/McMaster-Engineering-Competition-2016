README

This project simulates a weather system that broadcasts weather information and alerts sailors of critical/dangerous conditions.

Thus far, it does so in a discrete time step. Whenever the uesr presses "Start", the program simulates 1 time step.

To run the file:

1. Download and Extract MEC-Team1-Code.zip

2. Double click on Competition

3. Go to Dist

4. Double click on Competition.jar

5. To simulate a SINGLE time step, press Start

6. To enter test values for sensors, press test

7. Enter test values in the fields.

8. Press "Test" to run a time step with those values.

9. The report is available in the Google Drive.

10. JavaDocs are available in dist->javadoc.

11. Test cases are available in SystemTest.java.
